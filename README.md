## Run and test the app
First, clone the repo.

Then, run:

`npm i`

To **run** the app, run:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To **run tests**, run:

`npm run test`
